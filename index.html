autonomous robot

I designed and built this robot with parts from a Makeblock kit and fabricating acrylic and sheet metal. It was coded to navigate using a RC controller and perform tasks autonomously such as picking and dropping materials, placing flat items on a vertical surface, placing objects on the robot itself, etc. It was programmed in C++ and Scratch.

Skills Practiced:

  • CAD using Fusion360
  • Circuit design
  • Mechanical design & analysis
  • Programming a RC controller
  • Implementing autonomous algorithms

plantie

This is a plant growth aid and data collector that uses image proccessing to identify plants or Google's Speech-to-Text API if the user already knows their plant. It senses moisture and has scheduled LEDs that change colors to reflect the time and duration of outdoor light. It calculates growth rate of a user's plant and strives to optimize it! It was programmed in C++ and Python.

Skills Practiced:

  • Programming scheduled functions in Arduino
  • Programming with both cores of a dual-core processor
  • Implementing speech to text
  • Sending information back & forth between a server & an ESP32
  • Working with databases
  • Circuit design

moon light

This is a lamp I made for a friend who really enjoys photographing the moon. It gathers data from a weather and moon ph ases API that it checks daily and changes color depending on what optimal phase for photographing it is, turquoise is a full moon, light blue for a supermoon, red for a total eclipse and pink for a partial one. Depending on when the moon rises and sets everyday, the light blinks for a minute leading up to those times. It was programmed in C++ and Python.

Skills Practiced:

  • Programming scheduled functions in Python
  • Using an Arduino ESP32 to periodically check time
  • Sending information back & forth between a server & an ESP32
  • Working with databases
  • Circuit design
